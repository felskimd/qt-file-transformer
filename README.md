Для маски ипользуются регулярные выражения, разделённые пробелом. 
Например, *.bin *.txt или просто * для выбора всех файлов (не рекомендуется).

Входные файлы должны располагаться в папке с исполняемым файлом.

Пути и названия документов не должны содержать кириллических символов.

Если так получилось, что включена перезапись выходных файлов и их директория совпадает с директорией входных файлов, входные файлы будут очищены.
